A standalone resource pack with the custom gear and item textures from **[VanillaSkills](/modhub/mods/vanillaskills/)** — the Hardwood, Rose Gold, Steel, Crystalline and Dragon armor and tools, the Dragon Ingot and Scale, and the rest.

If you run VanillaSkills you **already receive these automatically** (the server pushes the pack to every player on join). This standalone pack is for using the look on its own, or bundling it into a modpack's resource packs.

## Requirements

- Minecraft **26.2** (any loader — it's a resource pack)

## Download

- [CurseForge](https://www.curseforge.com/minecraft/texture-packs/vanilla-skills-texture-pack)

## Notes

- The pack is kept in lockstep with the mod's own bundled textures and language files, so item and gear names show correctly (including translations) even on vanilla clients.
- On a modded client, VanillaSkills' pushed pack takes care of this for you — you only need this standalone pack if you want the textures without the mod.
